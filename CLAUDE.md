# es-typed-go

## Commands

```bash
# Unit tests
go test ./... -short -count=1

# Integration tests (start Elasticsearch first: docker compose up -d)
go test -tags=integration -v -timeout=120s ./estype/... ./esv8/...  # v8 on port 19200
go test -tags=integration -v -timeout=120s ./esv9/...               # v9 on port 19201

# Regenerate esclient_spec.go and api_coverage_test.go
go generate ./...

# Generate estype.Field constants from a mapping file or Go struct
go tool estyped -mapping mapping.json -out model.go -package model
go tool estyped -mapping mapping.json -out model.go -package model -group Sample  # grouped mode
go tool estyped -struct MyType -out model.go  # -package defaults to $GOPACKAGE
```

## Testing

- Assertion library: `gotest.tools/v3/assert` (`assert.NilError`, `assert.Equal`, …)
- `t.Parallel()` at the top of every test and every sub-test
- Integration tests require `//go:build integration` build tag
- Field names in tests must be generic only: `status`, `title`, `category`, `tags`, `items`, `date`, `price`, `id`, `name`, `type`, `enabled`, `value`
- Integration test ES URL: `http://localhost:19200` (v8) / `http://localhost:19201` (v9); override with `ES_URL` env var

## Architecture decisions

**v8 ↔ v9 parity**: `esv8/` and `esv9/` are mirrors — changes to one typically need the same change in the other.

**ESClient vs ESClientSpec**: `ESClient` is the curated interface for application code. `ESClientSpec` is fully generated from the ES spec; use it only when `ESClient` lacks the needed endpoint.

**Search[T] vs SearchRaw**: `Search[T](ctx, client, alias, params)` is the preferred API and decodes `_source` into T. `SearchRaw` is an escape hatch for request shapes not covered by `SearchParams`.

**SearchBuilder output**: `query.NewSearch().Build()` returns `query.SearchParams`. Map it to `esv8.SearchParams` or `esv9.SearchParams` before passing to `Search[T]`.

## Generated files

Do not edit manually — regenerate with `go generate ./...`:
- `esv8/esclient_spec.go`, `esv9/esclient_spec.go`
- `esv8/api_coverage_test.go`, `esv9/api_coverage_test.go`

Header for `go generate`-produced files: `// Code generated by "go generate"; DO NOT EDIT.`
Header for `estyped`-produced files: `// Code generated by estyped; DO NOT EDIT.`
